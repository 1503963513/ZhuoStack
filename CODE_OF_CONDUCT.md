# 行为准则

## 我们的承诺

为了营造开放、友善和安全的社区环境，项目维护者和参与者承诺：无论年龄、身体状况、民族、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向如何，参与项目的每个人都应获得尊重。

## 我们期望的行为

- 使用友善、包容和专业的语言；
- 尊重不同的观点、经验和技术选择；
- 接受建设性的批评，聚焦代码、文档和事实；
- 对错误负责，并从中学习；
- 关注社区整体利益，而不是个人胜负。

## 不可接受的行为

- 性化语言、图像或主动示好，以及不受欢迎的性关注；
- 恐吓、骚扰、攻击、贬损或人身指向；
- 发布他人的私人信息或凭据；
- 未经许可发布私下沟通内容；
- 其他在专业社区中合理地被视为不当或不可接受的行为。

## 执行与报告

请通过 [SECURITY.md](SECURITY.md) 中的私密渠道报告骚扰、隐私泄露或其他行为准则问题，不要在公开 Issue 中发布当事人的敏感信息。维护者会审核报告、保护报告人的隐私，并根据情节采取警告、删除内容、暂时限制参与或永久移除等措施。

维护者有权为不符合本行为准则的评论、提交、Issue 或其他贡献采取适当措施，并会尽量说明执行决定。

本行为准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。

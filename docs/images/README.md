# ZhuoStack 项目截图

此目录用于存放 README、GitHub 和 Gitee 项目页使用的截图、架构图和演示动图。

建议文件命名：

- `hero.webp`：README 首屏主视觉图
- `login.webp`：登录页
- `user-management.webp`：用户管理
- `rbac-permissions.webp`：角色与权限
- `file-storage.webp`：文件管理与上传
- `system-monitoring.webp`：系统监控
- `swagger-api.webp`：Swagger API 文档
- `ci-pipeline.webp`：CI 质量门禁
- `architecture.svg`：系统架构图
- `demo.gif`：简短操作演示

优先使用相对路径引用图片，例如：

```md
![ZhuoStack 用户管理](docs/images/user-management.webp)
```

请勿提交真实密码、Token、云服务密钥、个人数据、内网地址或生产环境截图。
